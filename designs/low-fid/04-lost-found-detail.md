## Interactions

### Click Actions
- **Back Button** → Return to list view
- **Image** → Open lightbox (enlarged view)
- **Contact Owner** → Open contact modal
- **Edit Post** → Navigate to edit form
- **Mark Resolved** → Open confirmation modal
- **Delete Post** → Open delete confirmation
- **Similar Item Card** → Navigate to that item's detail
- **User Avatar** → View user profile

### Hover States
- **Image**: Show "Click to enlarge" overlay
- **Contact Button**: Background color change
- **Similar Items**: Slight elevation
- **Action Buttons**: Color change

## Responsive Behavior

- **Desktop**: Two-column layout (image + details)
- **Tablet**: Two-column layout (narrower)
- **Mobile**: Single column (stacked)

---

## Accessibility

- **Alt Text**: All images have descriptive alt text
- **ARIA Labels**: Buttons have clear labels
- **Keyboard Nav**: Tab through all interactive elements
- **Screen Reader**: Proper heading hierarchy
- **Focus States**: Visible focus indicators
