Demo video guidance

- Create a 60–90 second demo showing:
  - Quick sign-up / login (Google OAuth + email path)
  - Upload a profile picture and create a lost/found report
  - Show chat: send and receive a message
  - Browse campus map and open an event detail

- Recording tips:
  - Use 1280x720 resolution, MP4 (H.264) codec
  - Keep narration concise and highlight key flows
  - Upload the final `demo.mp4` to this repo's `docs/` folder or host on YouTube and link here

- Add the recording to README with a short embed or link.
