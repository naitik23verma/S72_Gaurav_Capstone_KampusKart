## Interactions

### Click Actions
- **Cancel** → Show unsaved changes warning (if dirty)
- **Post Item** → Validate form → Submit → Show success
- **Upload Area** → Open file picker
- **Image X** → Remove image with confirmation
- **Category Dropdown** → Show category options
- **Date/Time Picker** → Open picker modal
- **Back Button** → Navigate back with warning

### Keyboard Shortcuts
- **Ctrl/Cmd + Enter** → Submit form
- **Esc** → Cancel (with warning)
- **Tab** → Navigate fields

### Auto-save (Optional)
- Save draft every 30 seconds
- Show "Draft saved" indicator
- Restore draft on return

---

## Form Validation Rules

- **Title**: Required, 5-100 characters
- **Description**: Required, 10-500 characters
- **Category**: Required, must select one
- **Status**: Required, Lost or Found
- **Location**: Optional, max 100 characters
- **Date/Time**: Optional, cannot be future
- **Images**: At least 1 required, max 5, max 5MB each
- **Contact**: Optional, valid email or phone format

---

## Accessibility

- **Labels**: All fields have clear labels
- **Required**: Marked with * and aria-required
- **Errors**: Announced to screen readers
- **Focus**: Visible focus indicators
- **Keyboard**: Full keyboard navigation
- **ARIA**: Proper ARIA attributes
