## 🎨 Design System

### Colors
- **Primary**: #2196F3 (Blue) - Trust, reliability
- **Secondary**: #4CAF50 (Green) - Success, found items
- **Semantic**: Success, Warning, Error, Info
- **Gray Scale**: #FAFAFA to #212121 (9 shades)
- **Category**: 9 colors for item categories

### Typography
- **Font**: System font stack
- **Sizes**: 12px to 48px (8 scales)
- **Weights**: Regular (400), Semibold (600), Bold (700)
- **Line Heights**: 1.2 to 1.6

### Spacing
- **Base Unit**: 8px
- **Scale**: 4, 8, 16, 24, 32, 40, 48, 64px

### Components
- Buttons (Primary, Secondary, Text)
- Inputs (Text, Textarea, Dropdown)
- Cards (Item, Stats, Update)
- Badges (Status, Category)
- Modals, Navigation, Footer
