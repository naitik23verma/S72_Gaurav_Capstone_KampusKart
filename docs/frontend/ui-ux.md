# UI and UX Standards

The primary UI system is documented in UI_UX_STANDARDS.md. This summary highlights how those standards show up in the app.

## Color system
- Primary: #181818 for buttons and headers
- Accent: #00C6A7 for hover and focus states
- Secondary: #F05A25 for destructive actions

## Typography
- Work Sans as the primary font
- Clear hierarchy for titles, sections, and labels

## Layout
- Consistent padding and spacing across pages
- Card based layout with rounded corners and 2px borders
- Responsive grids for feature lists

## Interaction patterns
- Hover and active feedback on cards and buttons
- Clear empty states and error banners
- Accessibility with aria labels and keyboard navigation

## Loading states
- Skeleton loaders for pages, lists, and profile
- Button spinners for actions

For detailed standards, see UI_UX_STANDARDS.md in the repo root.
