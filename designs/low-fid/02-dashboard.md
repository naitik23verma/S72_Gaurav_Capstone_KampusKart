---

## Interactions

### Hover States
- **Item Cards**: Slight elevation, border highlight
- **Stats Cards**: Background color change
- **Update Cards**: Background color change
- **FAB**: Scale up slightly

### Click Actions
- **Item Card** → Navigate to detail view
- **View All** → Navigate to full list
- **Stats Card** → Filter dashboard by stat
- **Update Card** → Expand full update
- **FAB** → Open create item form

### Loading State
```
┌────────────────────────────┐
│                            │
│    [Spinner Animation]     │
│    Loading dashboard...    │
│                            │
└────────────────────────────┘
```
