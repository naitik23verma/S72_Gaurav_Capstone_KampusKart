# Low-Fidelity Wireframes ✅

ASCII-based wireframes documenting all core user flows for KampusKart.

**Created**: Day 2 (January 16, 2026)  
**Format**: ASCII art wireframes (version-control friendly)  
**Coverage**: 15+ screens across 5 user flows

---

## 📁 Files

1. **LOW_FID_FLOW.md** - Overview, user scenarios, and navigation structure
2. **01-auth-flow.md** - Landing, registration, login, and OAuth flows
3. **02-dashboard.md** - Main dashboard with stats, recent items, and updates
4. **03-lost-found-list.md** - Browse, search, filter, and pagination
5. **04-lost-found-detail.md** - Item details, contact, and owner actions
6. **05-create-edit-form.md** - Create/edit forms with validation

---

## 🎯 User Flows Covered

- ✅ Authentication (register, login, OAuth)
- ✅ Dashboard (stats, recent items, updates)
- ✅ Lost & Found List (search, filter, browse)
- ✅ Item Detail (view, contact, edit, delete)
